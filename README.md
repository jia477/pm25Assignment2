# URBAN5160 PM2.5 Assignment 2

This repository contains my URBAN5160 assignment on predicting PM2.5 across London Output Areas using graph neural networks.

## Files
- `exercise2_pm25_gnn_clean.ipynb` — source notebook
- `exercise2_pm25_gnn_clean.html` — final HTML notebook for submission

## Summary
The project treats Output Areas as nodes, builds a centroid-based k-nearest neighbours graph, and compares two-layer GNN models with a simple baseline.

## Outputs
The notebook includes data exploration, graph construction, GNN modelling, RMSE and R², a prediction vs actual plot, a predicted PM2.5 map, and a short discussion of model performance.
